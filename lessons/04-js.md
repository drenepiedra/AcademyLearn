## Introducción a JavaScript

JavaScript (JS) es un lenguaje de programación ligero, interpretado (o compilado justo a tiempo) con funciones de primera clase. Si bien es más conocido como lenguaje de scripting para páginas web, muchos entornos no relacionados con navegadores también lo utilizan, como Node.js, Apache CouchDB y Adobe Acrobat. JavaScript es un lenguaje dinámico, basado en prototipos y con recolección de basura, compatible con múltiples paradigmas, como el imperativo, el funcional y el orientado a objetos.
